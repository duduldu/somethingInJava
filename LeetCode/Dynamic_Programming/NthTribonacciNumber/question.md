泰波那契序列Tn定义如下：
T0 = 0, T1 = 1, T2 = 1, 且在 n >= 0的条件下 Tn+3 = Tn + Tn+1 + Tn+2
给你整数n，请返回第 n 个泰波那契数Tn 的值。

示例 1：
输入：n = 4
输出：4
解释：
T_3 = 0 + 1 + 1 = 2
T_4 = 1 + 1 + 2 = 4

示例 2：
输入：n = 25
输出：1389537

Given n, return the value of Tn.

Example 1:

Input: n = 4
Output: 4
Explanation:

T_4 = 1 + 1 + 2 = 4

Example 2:
